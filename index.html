Introduction to Web Development
  • What is Web Development?
  • Frontend vs Backend
  • How websites work (Browser – Server – Database)
  • Objective: Students learn how to create the structure of a webpage.
1. Introduction to HTML (10 mins)
  • What is HTML?
  • Structure of a webpage
  • Basic HTML page structure
2. Basic Tags (15 mins)
  • Headings (h1–h6)
  • Paragraph, Line break, Horizontal line
3. Formatting Tags (10 mins)
  • Bold, Italic, Underline
  • Lists (ordered & unordered)
4. Links & Images (10 mins)
  • Anchor tag & Image tag
  • Attributes (href, src, alt)
5. Forms Basics (10 mins)
  • Input types, Label, Button
  • Simple registration form structure
6. Mini Task (5 mins)
  • Create a simple webpage with Heading, Paragraph, Image, Link.
1. Introduction to CSS (10 mins)
  • What is CSS?
  • Types of CSS: Inline, Internal, External
2. Selectors & Colors (15 mins)
  • ID selector
  • Text color & Background color
3. Fonts & Text Styling (10 mins)
  • font-size, font-family
  • text-align, text-decoration
4. Box Model (10 mins)
  • Margin, Padding, Border
5. Mini Task (5 mins)
  • Style the HTML page: Add colors, fonts, spacing, center content.
1. Introduction to Database (10 mins)
  • What is Database? What is RDBMS?
  • Real-time examples (College, Banking, E-commerce)
2. Introduction to SQL (5 mins)
  • What is SQL?
  • Types of SQL Commands (DDL, DML, DQL – overview only)
3. Creating Database & Table (15 mins – Live Demo)
  • CREATE DATABASE, CREATE TABLE
  • Data Types (INT, VARCHAR), PRIMARY KEY, AUTO_INCREMENT
4. CRUD Operations (20 mins – Practice)
  • INSERT, SELECT, UPDATE, DELETE
  • WHERE condition
5. Mini Task (10 mins)
  • Create a simple table, Insert 2–3 records, Display records with condition.
1. Introduction to PHP (10 mins)
  • What is Server-Side Scripting?
  • PHP vs HTML, How PHP Works
2. PHP Basics (15 mins)
  • PHP Syntax, Variables, Data Types
  • echo / print, Comments
3. Conditional Statements (10 mins)
  • if, else, else if
  • Comparison operators & programs
4. PHP + MySQL Connection (10 mins)
  • mysqli_connect()
  • Insert form data into database and view database
